# 창UP! 히히

"창UP! 히히"는 결제데이터 정보를 기반으로 사용자에게 상권 및 업종을 분석을 제공하고 적절한 상권 및 업종을 추천해주는 시스템 입니다.

## 설치 및 환경 설정

1. Git 저장소를 클론합니다.

```shell
git clone https://github.com/pupba/recommand-system
```

2. 필요한 종속성 설치

```shell
pip install -r requirements.txt
```

3. 서버를 실행

```shell
python server.py
```

-   데이터베이스는 적절히 만들어서 사용하세요!
